Control Flow

Conditionals, loops, comprehensions, and Pythonic iteration patterns.

3.1 Conditional Statements

Python uses if / elif / else with indentation-defined blocks. There are no braces or parentheses required around the condition.

temperature = 35

if temperature > 30:
    print("Hot")
elif temperature > 20:
    print("Warm")
else:
    print("Cool")

Ternary Expression

# C++/Java:  result = (x > 0) ? "positive" : "non-positive";
result = "positive" if x > 0 else "non-positive"
Python has no switch statement before 3.10. For older code, use a chain of elif or a dictionary dispatch pattern.

3.2 Loops

for Loop

Python's for iterates over any iterable — lists, strings, ranges, files, etc. It does not use an index counter by default like C-style for loops.

languages = ["Python", "C++", "Java"]
for lang in languages:
    print(lang)

# Index-based iteration (if you need the index)
for i, lang in enumerate(languages):
    print(f"{i}: {lang}")

while Loop

count = 5
while count > 0:
    print(count)
    count -= 1   # No -- operator in Python

break, continue, and Loop else

for n in range(2, 20):
    for d in range(2, n):
        if n % d == 0:
            break
    else:
        # Executes when the loop completes WITHOUT break
        print(f"{n} is prime")
The else clause on loops is unique to Python and has no equivalent in C++/Java. It runs only when the loop terminates normally (not via break).

3.3 Structural Pattern Matching

Python 3.10 introduced match/case, which is more powerful than C++/Java switch — it supports destructuring, guards, and wildcard patterns.

def describe_http(status):
    match status:
        case 200:
            return "OK"
        case 404:
            return "Not Found"
        case 500:
            return "Server Error"
        case _:
            return f"Unknown: {status}"

3.4 Comprehensions

Comprehensions provide a concise way to build collections from iterables. They replace many common for-loop patterns.

# List comprehension
squares = [x ** 2 for x in range(10)]

# With filter
evens = [x for x in range(20) if x % 2 == 0]

# Dict comprehension
word_lengths = {w: len(w) for w in ["hello", "world"]}

# Set comprehension
unique_lengths = {len(w) for w in ["cat", "dog", "ant"]}

Nested Comprehensions

# Flatten a 2D matrix
matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
flat = [val for row in matrix for val in row]
# [1, 2, 3, 4, 5, 6, 7, 8, 9]

3.5 Iteration Utilities

# zip - iterate over parallel sequences
names = ["Alice", "Bob"]
scores = [95, 87]
for name, score in zip(names, scores):
    print(f"{name}: {score}")

# range with step
for i in range(0, 20, 3):
    print(i)   # 0, 3, 6, 9, 12, 15, 18

# reversed and sorted
for item in reversed(sorted([3, 1, 4, 1, 5])):
    print(item)
