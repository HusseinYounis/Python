5.1 Lists

A list is a mutable, ordered sequence — similar to std::vector<T> in C++ or ArrayList<T> in Java, but heterogeneous (elements can differ in type).

nums = [10, 20, 30, 40, 50]

# Indexing (0-based, same as C++/Java)
print(nums[0])      # 10
print(nums[-1])     # 50 (last element)

# Slicing [start:stop:step]
print(nums[1:4])    # [20, 30, 40]
print(nums[::2])    # [10, 30, 50]
print(nums[::-1])   # [50, 40, 30, 20, 10] reversed

Common List Operations

nums.append(60)           # Add to end - O(1)
nums.insert(0, 5)         # Insert at index - O(n)
nums.extend([70, 80])     # Append iterable
val = nums.pop()          # Remove & return last - O(1)
nums.remove(30)           # Remove first occurrence - O(n)
nums.sort()               # In-place sort - O(n log n)
sorted_copy = sorted(nums) # Return new sorted list

Shallow vs Deep Copy

import copy

a = [[1, 2], [3, 4]]
b = a.copy()              # Shallow copy
c = copy.deepcopy(a)      # Deep copy

a[0][0] = 99
print(b[0][0])  # 99 - shared inner list
print(c[0][0])  # 1  - independent copy

5.2 Tuples

Tuples are immutable sequences. Once created, elements cannot be changed. They are hashable if all elements are hashable and can be used as dictionary keys.

point = (3.0, 4.0)
x, y = point              # Tuple unpacking

# Swap without temp variable
a, b = 1, 2
a, b = b, a                # a=2, b=1

# Named tuples for structured data
from collections import namedtuple
Student = namedtuple("Student", ["name", "gpa", "major"])
s = Student("Alice", 3.9, "CE")
print(s.name, s.gpa)       # Alice 3.9

5.3 Dictionaries

Dictionaries are hash maps — equivalent to std::unordered_map in C++ or HashMap in Java. Average O(1) lookup, insertion, and deletion.

student = {
    "name": "Alice",
    "gpa": 3.9,
    "courses": ["CS101", "MATH201"]
}

# Access
print(student["name"])              # Alice
print(student.get("age", "N/A"))    # N/A (safe access)

# Mutation
student["age"] = 21                 # Add/update
del student["gpa"]                  # Remove key

# Iteration
for key, value in student.items():
    print(f"{key}: {value}")

defaultdict and Counter

from collections import defaultdict, Counter

# defaultdict auto-creates missing keys
word_count = defaultdict(int)
for word in "the cat sat on the mat".split():
    word_count[word] += 1

# Counter - specialized frequency counter
freq = Counter("abracadabra")
print(freq.most_common(3))  # [('a', 5), ('b', 2), ('r', 2)]

5.4 Sets

Sets are unordered collections of unique, hashable elements — similar to std::unordered_set in C++ or HashSet in Java.

a = {1, 2, 3, 4, 5}
b = {4, 5, 6, 7, 8}

print(a | b)    # Union:        {1, 2, 3, 4, 5, 6, 7, 8}
print(a & b)    # Intersection: {4, 5}
print(a - b)    # Difference:   {1, 2, 3}
print(a ^ b)    # Symmetric:    {1, 2, 3, 6, 7, 8}

# Fast membership test - O(1)
print(3 in a)   # True
