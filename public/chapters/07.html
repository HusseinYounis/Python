Modules, Packages, and the Standard Library

Organizing code with modules, creating packages, pip, and essential stdlib modules.

7.1 Importing Modules

A module is simply a .py file. Python's import system is similar to Java's import but more flexible, and different from C++'s #include which is textual substitution.

import math                        # Import entire module
from math import sqrt, pi          # Import specific names
from math import factorial as fact  # Alias
import numpy as np                  # Common alias convention

The __name__ Guard

# utils.py
def helper():
    return "I help"

if __name__ == "__main__":
    # Only runs when executed directly, not when imported
    print(helper())
This is analogous to Java's public static void main and C++'s int main() convention, but more flexible since any module can be both importable and executable.

7.2 Creating Packages

A package is a directory containing an __init__.py file (can be empty). Subpackages are nested directories.

myproject/
    __init__.py
    core/
        __init__.py
        engine.py
        config.py
    utils/
        __init__.py
        math_helpers.py
        io_helpers.py
# Importing from packages
from myproject.core.engine import Engine
from myproject.utils import math_helpers

# Relative imports (within the package)
from .config import DEFAULT_PORT        # Same package
from ..utils.io_helpers import read_csv # Parent package

7.3 pip and Dependency Management

# Install a package
pip install requests

# Install specific version
pip install numpy==1.24.0

# Install from requirements file
pip install -r requirements.txt

# Export current environment
pip freeze > requirements.txt

# Uninstall
pip uninstall requests

7.4 Essential Standard Library Modules

pathlib - File System Paths

from pathlib import Path

p = Path("data") / "input.csv"    # OS-independent path joining
print(p.exists())                  # True/False
print(p.suffix)                    # .csv
print(p.stem)                      # input

for f in Path(".").glob("**/*.py"):
    print(f)

json - JSON Serialization

import json

data = {"name": "Alice", "scores": [95, 87, 92]}

# Serialize to string
json_str = json.dumps(data, indent=2)

# Deserialize from string
parsed = json.loads(json_str)

# File I/O
with open("data.json", "w") as f:
    json.dump(data, f)
with open("data.json") as f:
    loaded = json.load(f)

datetime — Date and Time

The datetime module provides classes for manipulating dates and times. The four key classes are:

  • date — a calendar date (year, month, day)
  • time — a time of day (hour, minute, second, microsecond)
  • datetime — combines date and time into one object
  • timedelta — a duration (difference between two dates/times)

Creating Date and Time Objects

from datetime import date, time, datetime, timedelta

# Current date/time
today = date.today()                    # e.g. 2025-03-15
now = datetime.now()                    # e.g. 2025-03-15 14:30:00.123456

# Create specific dates
birthday = date(2000, 5, 17)            # May 17, 2000
meeting = datetime(2025, 6, 1, 9, 30)   # June 1, 2025 at 09:30

# Create from string (parsing)
d = date.fromisoformat("2025-12-25")
dt = datetime.fromisoformat("2025-12-25T18:00:00")

Extracting Components

from datetime import datetime

now = datetime.now()

# Extract individual components
print(now.year)        # 2025
print(now.month)       # 3
print(now.day)         # 15
print(now.hour)        # 14
print(now.minute)      # 30
print(now.second)      # 0

# Get just the date or time part
print(now.date())      # 2025-03-15
print(now.time())      # 14:30:00.123456

# Day of week (0=Monday, 6=Sunday)
print(now.weekday())   # 5 (Saturday)

# ISO day of week (1=Monday, 7=Sunday)
print(now.isoweekday())  # 6

# ISO calendar (year, week number, weekday)
print(now.isocalendar())  # (2025, 11, 6)

Formatting and Parsing

from datetime import datetime

now = datetime.now()

# Format to string with strftime
print(now.strftime("%Y-%m-%d"))           # "2025-03-15"
print(now.strftime("%d/%m/%Y"))           # "15/03/2025"
print(now.strftime("%B %d, %Y"))          # "March 15, 2025"
print(now.strftime("%H:%M:%S"))           # "14:30:00"
print(now.strftime("%I:%M %p"))           # "02:30 PM"
print(now.strftime("%A, %B %d, %Y"))      # "Saturday, March 15, 2025"

# Parse string to datetime with strptime
dt = datetime.strptime("25/12/2025", "%d/%m/%Y")
print(dt)   # 2025-12-25 00:00:00

# f-string formatting (Python 3.6+)
print(f"Today is {now:%A, %B %d}")        # "Today is Saturday, March 15"

Date Arithmetic with timedelta

Use timedelta to add or subtract durations from dates:

from datetime import date, datetime, timedelta

today = date.today()

# Add days
tomorrow = today + timedelta(days=1)
next_week = today + timedelta(weeks=1)
deadline = today + timedelta(days=30)
print(f"Deadline: {deadline}")       # 30 days from now

# Subtract days
yesterday = today - timedelta(days=1)
last_month = today - timedelta(days=30)

# Add hours/minutes to datetime
now = datetime.now()
in_2_hours = now + timedelta(hours=2)
in_90_min = now + timedelta(minutes=90)
print(f"Meeting at: {in_2_hours:%H:%M}")

# Combine multiple units
future = now + timedelta(days=5, hours=3, minutes=30)

# Subtract two dates to get a timedelta
birthday = date(2025, 12, 25)
days_left = birthday - today
print(f"Days until birthday: {days_left.days}")  # e.g. 285
print(type(days_left))  # <class 'datetime.timedelta'>

Comparing Dates

from datetime import date, datetime

d1 = date(2025, 1, 1)
d2 = date(2025, 12, 31)

print(d1 < d2)      # True
print(d1 == d2)      # False
print(d1 != d2)      # True

# Sort a list of dates
dates = [date(2025, 3, 15), date(2024, 12, 1), date(2025, 1, 20)]
print(sorted(dates))
# [datetime.date(2024, 12, 1), datetime.date(2025, 1, 20), datetime.date(2025, 3, 15)]

Practical Example: Age Calculator

from datetime import date

def calculate_age(birth_date):
    today = date.today()
    age = today.year - birth_date.year
    # Subtract 1 if birthday hasn't occurred yet this year
    if (today.month, today.day) < (birth_date.month, birth_date.day):
        age -= 1
    return age

birthday = date(2000, 5, 17)
print(f"Age: {calculate_age(birthday)} years")   # e.g. "Age: 24 years"

# Days alive
days_alive = (date.today() - birthday).days
print(f"Days alive: {days_alive:,}")              # e.g. "Days alive: 9,068"
Tip: The datetime module handles all common date/time operations. For time zones use datetime.timezone or the third-party pytz/zoneinfo library. For more advanced date arithmetic (e.g., “add 1 month”), see the dateutil package (pip install python-dateutil).
