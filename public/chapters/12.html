12.1 Creating Arrays

NumPy's ndarray is a contiguous block of typed memory — conceptually like a C++ array but with Python convenience. Unlike Python lists, all elements share the same type.

import numpy as np

# From lists
a = np.array([1, 2, 3, 4, 5])
b = np.array([[1, 2, 3], [4, 5, 6]])  # 2D

# Factory functions
zeros = np.zeros((3, 4))              # 3x4 of zeros
ones = np.ones((2, 3), dtype=np.int32)
eye = np.eye(4)                        # 4x4 identity
rng = np.arange(0, 10, 0.5)           # Like range() but float
lin = np.linspace(0, 1, 100)          # 100 points in [0, 1]

print(b.shape)   # (2, 3)
print(b.dtype)   # int64
print(b.ndim)    # 2

12.2 Vectorized Operations

Vectorized operations execute in compiled C code, avoiding slow Python loops. This is the key to NumPy's performance.

a = np.array([1, 2, 3, 4, 5])

# Element-wise operations (no loops!)
print(a * 2)          # [ 2  4  6  8 10]
print(a ** 2)         # [ 1  4  9 16 25]
print(np.sqrt(a))     # [1.   1.41 1.73 2.   2.24]

# Aggregations
print(a.sum())        # 15
print(a.mean())       # 3.0
print(a.std())        # 1.414
print(a.max())        # 5
print(a.argmax())     # 4 (index of max)
Rule of thumb: if you're writing a Python for loop over array elements, there's probably a vectorized NumPy function that does it 10-100x faster.

12.3 Indexing and Slicing

m = np.array([[10, 20, 30],
              [40, 50, 60],
              [70, 80, 90]])

print(m[1, 2])        # 60 (row 1, col 2)
print(m[:, 1])        # [20 50 80] (all rows, col 1)
print(m[0:2, :])      # First two rows

# Boolean indexing (masking)
print(m[m > 50])      # [60 70 80 90]

# Fancy indexing
rows = [0, 2]
cols = [1, 0]
print(m[rows, cols])  # [20 70]

12.4 Broadcasting

Broadcasting allows operations between arrays of different shapes by automatically expanding dimensions.

# Scalar broadcast
a = np.array([[1, 2, 3], [4, 5, 6]])
print(a + 10)    # [[11 12 13] [14 15 16]]

# Row vector broadcast
row = np.array([100, 200, 300])
print(a + row)   # [[101 202 303] [104 205 306]]

# Column vector broadcast
col = np.array([[10], [20]])
print(a + col)   # [[11 12 13] [24 25 26]]

12.5 Linear Algebra

A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

# Matrix multiplication
C = A @ B                      # or np.dot(A, B)
print(C)  # [[19 22] [43 50]]

# Transpose
print(A.T)  # [[1 3] [2 4]]

# Inverse, determinant, eigenvalues
inv_A = np.linalg.inv(A)
det_A = np.linalg.det(A)       # -2.0
eigvals, eigvecs = np.linalg.eig(A)

# Solve linear system Ax = b
b = np.array([5, 11])
x = np.linalg.solve(A, b)
print(x)  # [1. 2.]
